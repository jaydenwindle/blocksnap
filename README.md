# HolderSnap - Token snapshots made simple

### Dev Setup

#### Backend
1. Install dependencies
  ```
  $ poetry install
  ```
2. Run migrations
  ```
  $ poetry run python manage.py migrate
  ```
3. Run server
  ```
  $ poetry run python manage.py runserver
  ```

#### Frontend
1. Install dependencies
  ```
  $ yarn
  ```
2. Run app
  ```
  $ yarn start
  ```
